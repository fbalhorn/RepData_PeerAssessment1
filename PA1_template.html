Imputing missing values

1.: The count of missing values in the steps column is:

sum(is.na(activity_data$steps))

## [1] 2304

2./3.: We can now replace the na values in the step column with the calculated mean value for the respective interval. If the value is still NaN, we insert a zero. The new dataset containing the missing data filled in following the mentioned strategy is saved in activity_data_corrected:

#create copy of original dataset
activity_data_corrected<-activity_data

meanStepsPerInterval<-aggregate(activity_data$steps,list(activity_data$interval),mean,na.rm=TRUE)
rownames(meanStepsPerInterval)<-meanStepsPerInterval$Group.1

#replace values
i=0
for (i in 1:nrow(activity_data)){
  if(is.na(activity_data$steps[i])){
    activity_data_corrected$steps[i] <- subset(meanStepsPerInterval, Group.1 == activity_data[i,3],select=c(x))[[1]]
    }
}

##Are there differences in activity patterns between weekdays and weekends?

1.: The factor $typeDay contains the information, if we're dealing with a weekday or a weekend-day:

activity_data_corrected$typeDay <- ifelse(wday(as.Date(activity_data_corrected$date))>1 & wday(as.Date(activity_data_corrected$date))<7 ,"weekday", "weekend")
head(activity_data_corrected$typeDay)

## [1] "weekday" "weekday" "weekday" "weekday" "weekday" "weekday"

2.: The results are displayed in the panel plot below.

ggplot(activity_data_corrected, aes(x=interval,y=steps))+geom_line()+facet_grid(.~typeDay)

On weekdays, we see larger amount of performed steps during the early intervals per day, i.e. between interval 500 and 1000, compared to weekends. A possible interpretation might be, that people start their days later on weekends on average.
